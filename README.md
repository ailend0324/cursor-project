# 🎯 Cursor Project - 智能化工具集合

> 一个集成多种智能化工具的综合项目，专注于提升工作效率和用户体验。

## 🌟 核心亮点

### 🚀 **Share Page - 一键网页发布系统** ⭐️⭐️⭐️⭐️⭐️
**让分享变得像发朋友圈一样简单！**

- 📍 **位置**：`/share_page/`
- 🎯 **功能**：一键生成精美分享网页，无需编程基础
- ✨ **特色**：5种精美模板 + 自动化发布 + 响应式设计
- 🔗 **在线演示**：https://ailend0324.github.io/cursor-project/

**快速体验：**
```bash
cd share_page
./quick_examples.sh travel  # 生成旅行示例
./publish.sh                # 一键发布
```

## 🚀 快速开始

### 🎯 推荐：从Share Page开始
**最简单、最实用的功能，3分钟即可体验！**

```bash
# 1. 进入share_page目录
cd share_page

# 2. 查看详细说明
cat README.md

# 3. 生成示例内容
./quick_examples.sh travel

# 4. 一键发布
./publish.sh

# 5. 2-3分钟后查看结果
# https://ailend0324.github.io/cursor-project/[你的页面名]/
```

### 📊 其他模块体验
```bash
# 智能客服系统
cd smart_service_agent
python scripts/data_processing/analyze_intents.py

# 教育模块
cd education
open index.html

# VPN工具
cd vpn
./vpn_monitor.sh
```
